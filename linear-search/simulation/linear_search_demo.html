• Enter the number you want to search for in the "Number To be Searched" box.
  • Click the Start button to begin the linear search demonstration.
  • The simulation will highlight each array element as it is checked.
  • If the searched number is found, it will be highlighted and the search will stop.
  • If the number is not found after checking all elements, the simulation will indicate that the element is not present in the array.
  • Adjust the slider to change the speed of the automatic search.
  • Click the Reset button to generate a new set of random numbers and start a new search.
